Лабораторна робота 4

Завдання на лабораторну роботу

  1. В робочій гілці власного репозиторію створити папку для виконання роботи
  2. Скопіювати в папку сайт з алораторної роботи 3
  3. Додати до сайту коментарі користувачів (з відповідним дизайном), які отримати з ресурсу Dummy JSON, використати параметри /posts, skip, limit.
  4. Значення skip взяти за номером у списку, limit = 5 для парних номерів у списку, 7 - для непарних
  5. Додати до сайту інформацію щодо погоди у місці проживання (з відповідним дизайном), яку отримати з ресурсу MeteoMatics.
  6. Використати технології html, scss (css), js (fetch API)
  7. Виконати commit, merge (гілка ready), push
  8. Підготуватися до захисту роботи
